# AutoCare-Tracker
Android vehicle maintenance tracking app

## Project Overview
AutoCare Tracker is an Android mobile application developed using Java. The app allows users to track and manage routine car and motorcycle maintenance tasks by storing service records digitally on the device.

## Problem Addressing
Vehicle maintenance is commonly tracked using paper logs or receipts, which are easy to lose and difficult to organize. AutoCare Tracker eliminates the need for paper maintenance logs by providing a centralized digital record that is always accessible on the userâ€™s Android device.

## Platform
- Android
- Android Studio
- Java

## Front-End / Back-End
### Front-End
- XML layouts
- Material Design components
- RecyclerView for task lists

### Back-End
- Java
- Room (SQLite) database
- ViewModel architecture components

## Application Functionality
- Add maintenance tasks
- Edit existing tasks
- Delete tasks
- Record service dates and mileage
- View maintenance history
- Optional notification reminders

## Design
The application includes the following screens:
- Home screen displaying maintenance tasks
- Add/Edit maintenance task screen
- Task detail screen

Wireframes are created prior to development to guide UI design.
